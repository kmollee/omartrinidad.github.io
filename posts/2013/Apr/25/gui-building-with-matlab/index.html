Why avoid GUIDE (GUI development environment)

With GUIDE is possible design UIs graphically, however instead of making it easier complicates it. In order to have a better control and understanding of the things I prefer create the GUI programmatically.

Create GUI programmatically

function varargout = mygui(varargin)
    % Brief description
    sizeScreen = get(0,'ScreenSize')
    hmf = figure('Position', sizeScreen,...
           'Name', 'Mammogram GUI',...
           'NumberTitle', 'off',...
           'MenuBar', 'none');
end

In the code above we have create a very basic window.
