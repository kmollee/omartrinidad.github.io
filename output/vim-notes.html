How to avoid use keyboard arrows

Use the keys h, j, k, and l is one of the more esoteric features of Vim, but also is one of the most powerful. In order to avoid use the normal arrow keys is possible lock them.

nnoremap <up> <nop>
nnoremap <down> <nop>
nnoremap <left> <nop>
nnoremap <right> <nop>
inoremap <up> <nop>
inoremap <down> <nop>
inoremap <left> <nop>
inoremap <right> <nop>
